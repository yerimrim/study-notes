- Operating System(OS) Services
    - Program creation, Program execution, Access to I/O devices, Controlled access to files, Systemp access, Error detection and response, Accounting
    - Process Management, Memory Management, File System Management, Device Management, User Interface(UI), Security and Access Control, Networking and Communication, Error Detection and Handling, Resource Allocation and Management
- Types of Operating Systems(OS)
    - **Interactive systems**: the user can interact directly with the computer.
    - **Batch multiprogramming**: the user cannot interact with the computer. The user only can see the results after the program is completed.
        - Maximize processor use.
        - Job control language commads provided with the job.
    - **Time sharing systems**: the user can interact directly with computer.
        - Minimize response time.
        - Commands enterned at the terminal.
