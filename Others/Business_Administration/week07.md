```jsx
<마케팅의 정의>
마케팅(market + ing): 시장에서 사고 파는 행위와 진행되는 상태
코틀러: "기업이 고객을 위해 가치를 창출하고 강력한 고객 관계를 구축함으로써 고객을 획득, 유지, 확대하는 전략"
드러커: "고객을 충분히 알고 이해함으로써 제품 또는 서비스를 적절하게 제공하고, 제품 또는 서비스가 스스로 판매되도록 만드는 것"

<마케팅 철학의 변천사>
'생산개념' -> '제품개념' -> '판매개념' -> '마케팅개념'

1. 생산개념: 대량 생산, 대량 유통으로 원가를 낮추고 가격을 낮춰야 한다는 철학.
=> 소비자 욕구의 다양화와 고도화로 인해 비효과적 개념이 됨.

2. 제품개념: 지속적인 제품 개선을 통해, 보다 우수한 제품을 개발하는 데 관심을 쏟는 철학.
=> 제품 개선이 R&D(Research & Development) 관점에서만 이뤄져, 시장에서 외면당할 수 있음.
=> 시장을 너무 좁게 보는 함정에 빠질 수 있음.
=> ex) 미국 철도 산업

3. 판매개념: 적극적 판매/촉진 노력이 없으면 소비자들이 충분한 양의 제품을 구매하지 않을 것이라는 사고.
(공급이 수요를 초과하는 상황에서 발생)
=> 초점이 고객 만족이 아니라 판매 증대에 맞춰져 있음 => 장기적 관점에서 성공 보장 x

4. 마케팅개념: 모든 마케팅 행위의 중심에 고객을 위치시키는 관리 철학.
=> 고객 욕구 파악, 경쟁사보다 고객의 욕구를 더 잘 충족시키기 위해 노력 (i.e. 제품에 맞는 고객을 찾는 것이 아닌, 고객에 맞는 제품을 찾는 것.)
=> 외부적 시각 (회사 밖에서부터 안으로)

<관여도>
: 어떤 대상에 대해 중요하게 생각하거나 관심을 갖는 정도
- 관여도에 따라 다른 마케팅 활동이 효력을 발휘함.
- 관여도가 높을수록, 많은 시간과 노력을 들여 정보를 탐색하고 대안을 평가함.
- 관여도는 상대적 (사람, 제품, 재무적 위험, 사회적 위험, 상황에 따라 달라짐)
- 관여도 높아지는 경우: 제품의 가격이 높음, 사회적 가시성이 높음, 구매 주기가 김, 지각된 위험이 큼.
- 관여도가 높은 경우 => 수치나 문자로 된 과학적 정보
- 관여도가 낮은 경우 => 그림이나 광고 모델과 같은 정보

<소비자 구매 의사결정 5단계>
문제의 인식 -> 정보의 탐색 -> 대안의 평가 -> 구매 -> 구매 후 행동

1.  문제의 인식
: 소비자가 '자신이 이상적으로 원하는 상태와 현재 상태에 차이가 있음'을 느끼는 단계
	이러한 문제의 인식은 내적 및 외적 요인에 의해 유발됨.
	(e.g.내적요인-생리적 욕구, 외적요인-타인의 물건이나 광고)

2. 정보의 탐색
: 문제를 해결할 수 있는 여러 대안에 대한 정보를 찾는 단계
	- 내적탐색: 기억에 저장된 정보를 떠올림.(e.g.환기상표군: 소비자가 특정 제품을 생각할 때, 바로 생각나는 상표)
	- 외적탐색: 광고, 친구, 인터넷 등을 통한 검색

3. 대안의 평가
: 소비자가 정보 탐색 단계에서 수집한 정보를 이용하여 대안을 평가하는 단계
	- 소비자는 제품의 여러 속성(기능, 디자인, 가격, 편리성, 내구성, 안정성 등)을 종합적으로 고려
	- 제품에 대한 관여도에 따라 평가기준의 수가 달라짐.
	- 기업은 표적 소비자가 어떤 평가 기준을 활용하는지, 어떤 방식으로 평가하는지를 파악하는 것이 중요함.

4. 구매
: 소비자가 평가결과에 따라 가장 선호하는 특정 상표의 제품을 구매하는 단계
	- 결정한 상표를 항상 구매하지는 않음.(소비자 자신의 변화, 새로운 대안의 출시 등 시장 상황의 변화로 인해)
	- 마케팅 노력은 구매 과정에도 이뤄져야 함.

5. 구매 후 행동
: 구매 이전에 제품에 대해 가졌던 기대와 구매 후 느끼는 제품 성과 간 차이가 만족도를 결정하는 단계
- 만족도는 재구매 및 신규 고객 유치 여부를 결정하는 중요한 요소. (e.g. 온라인 리뷰)
- 실제 구매 후 소비자는 자신의 의사결정이 옳았는지에 대한 확신의 부족으로 '인지부조화'(심리적불편)를 경험함.

Q) 1. 수요가 공급을 초과하는 상황에서는 마케팅 철학 중 '생산개념'이 유용하다.
	 2. 공급이 수요를 초과하는 상황에서는 마케팅 철학 중 '판매개념'이 유용하다.
	 3. 소비자들이 저렴한 제품을 선호한다는 가정에서 대량 생산과 유통을 통해 낮은 제품원가를 실현해야 한다는 관리 철학은 마케팅 철학 중 '생산개념'에 해당한다.

<STP 전략>
시장 세분화 -> 표적시장 선정 -> 포지셔닝

<1. 시장 세분화>
: 시장을 어떤 기준에 의해 여러 개의 '세분시장'으로 나누는 것.
	- 이질적인 소비자에 대해 동일한 방식의 접근은 비효율적임 => 시장 세분화가 필요함. (각 세분시장별로 서로 다른 마케팅 프로그램 제공)
	- 우선순위의 결정 => 한정된 자원의 효율적 사용
	- 시장 수요 변화에 신속한 대응

[시장 세분화의 기준]
- 인구통계학적 세분화 (나이, 성별, 소득, 교육수준 등)
- 지리적 세분화
- 심리적 세분화 (라이프스타일, 가치관, 관심분야 등)
- 편익 세분화 (같은 제품에 대해 중요시하는 요소. 운동화: 편안함, 디자인, 등)
- 사용량 세분화 (대량 소비자, 보통 사용자, 소량 소비자)

[바람직한 시장 세분화의 조건]
- 각 세분시장은 서로 이질적인 소비자 욕구를 가져야 함.
- 각 세분시장은 충분한 시장 규모를 가져야 함.
- 각 세분시장은 접근이 용이해야 함.
- 각 세분시장은 특징과 크기를 측정할 수 있어야 함.

<2. 표적시장 선정>
: 세분시장 중 어떤 세분시장을 목표로 할 것인지, 목표 세분시장을 몇 개로 할 것인지 정하는 것.
	- 표적시장의 수는 하나에서 전체까지도 가능, '순차적 진입' 가능

[표적시장 선정의 전략적 대안]
1) 비차별적 마케팅 전략
	- 시장을 하나의 커다란 시장으로 봄. (시장 세분화 개념 적용 x)
	- 소비자 간의 차이보다는 공통점에 중점.
	- 대다수의 구매자에게 어필할 수 있는 제품과 마케팅
	- 규모의 경제 가능, but 다양한 고객 욕구 충족에 한계
	- 다변화된 시장 환경에서는 부적절한 전략
	- e.g. 초기의 코카콜라, 아이폰

2) 차별적 마케팅 전략
	- 여러 개의 세분시장을 표적, 각각의 세분시장에 차별화된 전략으로 접근
	- 다양한 제품라인과 마케팅믹스를 통해 각 세분시장 안에서 강력한 지위 구축 가능
	- 많은 재원 필요(각가의 세분시장에 동시에 투자해야 하므로) => 규모가 큰 기업에 적합한 전략
	- e.g. 현대차, GAP

3) 집중적 마케팅 전략
	- 단일 혹은 소수의 한정된 세분시장을 표적
	- 소비자의 특성과 니즈를 하나하나 이해함. => 특화된 마케팅, 높은 고객 만족 가능
	- 자원이 제한된 기업이 주로 사용
	- 틈새시장: 세분시장을 더욱 세분화한 보다 작은 규모의 시장
	- e.g. 홀푸드마켓, 람보르기니, 블루보틀

[표적시장 선정 전략 선택 시 고려 요인]
1) 회사의 자원 (회사의 자원이 제한적 => 집중적 마케팅)
2) 제품의 변형성 (표준화된 제품 => 비차별적 마케팅) 
3) 제품 수명주기 단계 (도입기 => 비차별적 마케팅 or 집중적 마케팅)
4) 시장 다양성 (고객 욕구가 동질적 => 비차별 마케팅)
5) 경쟁사 전략 (경쟁사가 비차별적 마케팅 사용 => 차별적 마케팅 or 집중적 마케팅 선택)

<3. 포지셔닝>
: 표적 소비자의 마음 속에 자사 제품/브랜드의 차별적 우위를 구축하기 위한 노력.
	(포지션: 표적 소비자가 지각하고 있는 제품/브랜드의 주요 특징, 편익, 이미지 등)
	
[포지셔닝 방법]
결정된 제품 포지션은 제품 개발, 광고, 판촉 등과 같은 '마케팅믹스' 수단들의 통합적 관리를 통해 실현됨.	

[포지셔닝 선언문]
자사 제품/브랜드에 대해 간략히 설명 후, 해당 제품/브랜드가 표적시장의 특정 요구를 어떻게 충족시키는지 설명함.
=> 마케팅 노력을 회사의 브랜드 및 가치제안에 일치시키기 위한 것.

* 포지셔닝 맵: 소비자의 마음 속에 있는 자사 제품과 경쟁 제품의 위치를 2~3차원의 도면에 나타낸 것.

	 2. 세분시장 마케팅의 순서: 시장 세분화 -> 표적시장 선정 -> 포지셔닝
	 3. 기업이 시장세분화 전략을 펴는 이유: 유망한 세분시장을 공략해 한정된 자원을 효율적으로 사용할 수 있음.
```
