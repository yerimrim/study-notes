[인공지능이란]

<인공지능>
스스로 합리적인 판단을 할 수 있는 컴퓨터
알려준 것 이상을 처리할 수 있어야 함
알려준 정보만 활용하고 스스로 판단하지 않으면 인공지능이 아님
인공지능 기술의 핵심: 삶에 도움을 주고, 받을 수 있을까?

<인공지능의 유형>
1) 좁은 인공지능 (= 약한 인공지능)
- 한 가지에만 특화
- 합리적으로 생각하고 행동하는 시스템
- 한 가지 또는 특정한 영역에 국한된 인공지능
- ex. 알파고, 자율주행자동차, 애플의 시리, etc.

2) 일반 인공지능
- 인간 수준의 능력을 가진 인공지능
- 모든 분야에 적용될 수 있는 인공지능(일반화)

3) 슈퍼 인공지능
- 모든 면에서 인간보다 훨씬 뛰어난 지능을 가진 인공지능
- 과학적 창의력, 일반적인 지혜, 사회적 능력을 가짐

<인공지능AI, 머신러닝ML, 딥러닝DL의 관계>
: 인공지능 ⊃ 머신러닝 ⊃ 딥러닝
* AI 요소기술들을 더 잘 이해하기 위해 AI/ML/DL로 개념을 나눔 (분류 목적 x)

_________________________________________________________________________________________________________________________
[머신러닝이란]

<규칙기반시스템>
- 정교하게 설계된 Rule(Hand-designed Program)에 따라 결과를 출력
- 모든 과정을 사람이 직접 구성함
- 추론을 통해 가르쳐 주지 않은 질문에 답을 함
- ex. 동물 판별 시스템, Expert System(전문가의 사고 시스템을 질서정연한 프로그램으로 바꾼 것)
- Input -> hand-desigend program -> output

<(클래식)머신러닝>
- 컴퓨터 프로그래밍 데이터로부터 학습하는 과정
- 경험으로부터 지식을 학습
- 사람이 데이터의 특징을 결정하지만, 이후의 과정은 컴퓨터 스스로 진행
- Input -> hand-designed features -> mapping from features -> output

(hand-designed features): 사람이 사용된 데이터로부터 특징을 결정
(mapping from features): 컴퓨터가 hand-designed features를 활용하여 output을 출력

<머신러닝의 종류>
1) 지도학습: 답이 있는 학습, 레이블 o (분류 & 회귀)
2) 비지도학습: 같은 답이 없는 학습, 레이블 x
3) 강화학습: 행동의 시행착오를 통한 학습, **보상을 최대화**하는 방향으로 행위를 강화하는 학습 (ex. 자전거 타기)

_________________________________________________________________________________________________________________________
[딥러닝이란]

<표현학습>
: 데이터로부터 컴퓨터가 특징을 스스로 추출하여 학습 => 모든 과정을 컴퓨터가 관여
- Input -> features -> mapping from features -> output
- ex. 인공신경망 (매우 크고 다루기 어려운 데이터들의 특징을 모델링, 표현학습에 가장 적합한 구조)
- ex. 토끼와 고양이 구별, 손글씨 판별

<딥러닝>
: 고차원 특성(High-level Features)을 스스로 추출하여 학습
- 머신러닝의 한 분야
- 컴퓨터가 패턴을 인식하여 스스로 학습하도록 훈련시키는 기술
- 연속된 layer에서 점진적으로 의미 있는 표현을 배우는 데 강점이 있음
- Input -> **simple features** -> **additional layers of more abstract features** -> mapping from features -> output
- 다층구조(additional layers of more abstract features)의 장점: 결정경계를 자유롭게(비선형) 구축할 수 있음, 복잡한 Classification 문제의 정확도 향상
- ex. 사람 얼굴 인식
