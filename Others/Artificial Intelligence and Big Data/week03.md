[지식과 지식기반시스템]
Non-AI: 컴퓨터를 이용한 단순계산, 단순한 Computer Programming, 알려진 정보만 사용
AI: 스스로 생각하고 판단할 수 있는 컴퓨터, 알려진 것 이상 처리 가능 (AI ⊃ ML ⊃ RL ⊃ DL)
ML: 컴퓨터가 데이터로부터 학습하는 과정, 경험으로부터 지식 학습
	1) AI but not ML: 규칙기반 시스템(Rule-Based system), 전문가 시스템(Expert system)
	2) ML but not RL(Representation Learning, 표현학습 = Classical ML): Logistic Regression

Non-AI vs **Rule-Based AI** => 규칙기반 시스템은 알려주지 않은 사실도 출력 가능하기 때문
Rule-Based AI vs **ML** => 규칙기반 시스템은 복잡한 문제가 주어지면 시스템 복잡도가 증가하여 한계가 있음.
* 그럼에도 Robust 시스템을 위해서는 '사실에 근거한 추론'인 Rule-Based AI가 더 적합

데이터 -(처리)> 정보 -(일반화)> 지식
- 데이터: 활용가치가 낮음
- 정보: 데이터가 모여 만든 의미
- 지식: 추론을 위한 기본 정보

**모델링**: 문제 해결 시스템 구축
**정형화된 형태의 지식**: 지식을 논리, 의미망, 프레임, 규칙, 수학적 함수 등을 사용하여 표현한 것
**문제해결을 위한 모델 = 정형화된 형태의 지식 + 추론**
**지식기반 시스템**: 규칙기반 시스템(논리, 의미망, 프레임), 전문가 시스템(규칙, 그래프), 머신러닝 시스템(수학적 함수)

[지식표현과 추론]
논리: 말로 표현된 문장들에 대한 타당한 추론을 위한 것, 기호를 사용하여 문장 표현, 참 거짓 판정
		1) 명제 논리: 명제는 참 거짓을 분명하게 판정할 수 있는 문장
	 	2) 술어 논리: 술어는 대상의 속성이나 대상 간의 관계를 함수 형태로 나타낸 것

추론: 논리를 바탕으로 추론 (eg. 삼단논법)

규칙: IF, THEN 으로 표현되는 글
<규칙의 IF THEN 표현>          <규칙의 OR 그래프 표현>     <규칙의 AND 그래프 표현>
		IF -> THEN                     새이다.                  갈매기이다.
털이 있는 동물이다. -> 포유류이다.          /\                      /U\
									  	        날개가 있다. 날 수 있다.	    	 새이다. 바다 근처에 산다.				  	    
* IF THEN 보다 OR, AND 그래프 표현이 직관적이고 쉬움

의미망을 이용한 지식표현과 추론 = 인공지능
- 노드: 객체, 개념, 사건 (eg. 동물, 날기)
- 호: 노드 간의 관계 (can, cannot, has-a, is-a, do)
- 화살표: 낮은개념 -> 상위개념 (eg. 토끼 -> 초식)

의미망 표현의 장점: 
1) 시각적 지식표현으로 **직관적** 이해 가능
2) 용이한 지식 **추가** 및 **변경** 가능
3) 개념의 **계층관계**를 정의하여 속성의 상속관계 지정 가능
4) 복잡한 지식을 **구조화**하여 표현 가능

의미망 표현의 단점:
1) 지식의 양이 많아지면 관리 복잡
2) 개념이나 관계를 임의로 정의하기 때문에 **통일성 부족**
3) 논리적 결합 관계나 인과관계 기술시, 링크(AND, OR) 도입 필요 => 일관성이 떨어지고, 추론이 복잡해짐
4) 정적인 지식 표현

지식표현의 불확실성
- : 복잡한 대상/내용의 지식표현은 불확실한 점이 있다.
- 원인: 약한 관계성, 부정확한 언어 사용
- 퍼지 이론: 논리, 추론을 위해서는 이분적인 표현이 필요하지만, 모든 개념이나 번주가 항상 이분적이지 않음
					 따라서, 정도(degree)의 문제에는 퍼지집합(Fuzzy set) 도입

[전문가 시스템]
규칙 기반 시스템 = 추론을 통한 결과 출력: 지식(사실) + 추론규칙(IF,THEN,...)

추론을 통한 탐색
1) 순방향 추론: 규칙의 조건부 = 사실 (개수가 많아지면 연산 부담)
2) 역방향 추론: 결론부(가설) = 사실 (가설이 주어진 경우에 효과적)

<전문가 시스템>    **규칙기반 시스템 ⊃ 전문가 시스템, 전문가 시스템의 진단 != 전문가의 진단**
1) 추론엔진: 지식을 기반으로 정보를 추론 (정방향, 역방향 추론)
2) 지식베이스: 사실과 규칙을 나타냄 (지식 엔지니어 필요)
